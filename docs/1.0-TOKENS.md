# Design Tokens

## Colors

### Primary (Green)
| Token | Value | Usage |
|-------|-------|-------|
| `$color-primary` | `#58CC02` | Primary actions, success |
| `$color-primary-dark` | `#58A700` | 3D shadow, hover |
| `$color-primary-light` | `#89E219` | Hover state |

### Secondary (Blue)
| Token | Value | Usage |
|-------|-------|-------|

| `$color-secondary-dark` | `#1899D6` | 3D shadow |
| `$color-secondary-light` | `#49C0F8` | Hover state |
