# Components API

## Button

```html
<duo-button variant="primary" size="md">Click me</duo-button>
```

| Input | Type | Default | Description |
|-------|------|---------|-------------|
| `variant` | `'primary' \| 'secondary' \| 'danger' \| 'success' \| 'warning'` | `'primary'` | Color variant |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Button size |
| `disabled` | `boolean` | `false` | Disabled state |
| `fullWidth` | `boolean` | `false` | Full width button |
| `iconLeft` | `string` | - | Icon on left side |
| `iconRight` | `string` | - | Icon on right side |

| Output | Type | Description |
|--------|------|-------------|
| `clicked` | `EventEmitter<void>` | Emitted on click |

---

## Input

```html
<duo-input label="Email" placeholder="Enter email" size="md"></duo-input>
```

| Input | Type | Default | Description |
|-------|------|---------|-------------|
| `type` | `'text' \| 'password' \| 'email' \| 'number' \| 'tel' \| 'url' \| 'search'` | `'text'` | Input type |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Input size |
| `variant` | `'default' \| 'primary' \| 'secondary'` | `'default'` | Style variant |
| `label` | `string` | - | Label text |
| `placeholder` | `string` | - | Placeholder text |
| `hint` | `string` | - | Hint text below input |
| `error` | `string` | - | Error message |
| `disabled` | `boolean` | `false` | Disabled state |
| `fullWidth` | `boolean` | `false` | Full width input |

Implements `ControlValueAccessor` for form integration.

## Modal

```html
<duo-modal [(isOpen)]="showModal" title="Modal Title" size="md">
  <p>Content here</p>
  <div modal-footer>
    <duo-button (clicked)="showModal = false">Close</duo-button>
  </div>
</duo-modal>
```

| Input | Type | Default | Description |
|-------|------|---------|-------------|
| `isOpen` | `boolean` | `false` | Open/close state |
| `title` | `string` | - | Modal title |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Modal size |
| `showCloseButton` | `boolean` | `true` | Show X button |
| `closeOnBackdrop` | `boolean` | `true` | Close on backdrop click |
| `closeOnEscape` | `boolean` | `true` | Close on Escape key |
| `lockScroll` | `boolean` | `true` | Lock body scroll |

| Output | Type | Description |
|--------|------|-------------|
| `isOpenChange` | `EventEmitter<boolean>` | Two-way binding |
| `closed` | `EventEmitter<void>` | Emitted on close |

## Alert


| Input | Type | Default | Description |
|-------|------|---------|-------------|
| `variant` | `'info' \| 'success' \| 'warning' \| 'danger'` | `'info'` | Alert type |
| `title` | `string` | - | Alert title |
| `dismissible` | `boolean` | `false` | Show dismiss button |
| `showIcon` | `boolean` | `true` | Show variant icon |

| Output | Type | Description |
|--------|------|-------------|
| `dismissed` | `EventEmitter<void>` | Emitted on dismiss |

## Radio

```html
<duo-radio [options]="options" size="md" orientation="vertical"></duo-radio>
```

| Input | Type | Default | Description |
|-------|------|---------|-------------|
| `options` | `RadioOption[]` | `[]` | Radio options |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Radio size |
| `orientation` | `'horizontal' \| 'vertical'` | `'vertical'` | Layout direction |
| `disabled` | `boolean` | `false` | Disabled state |

```typescript
interface RadioOption {
  value: string | number;
  label: string;
  disabled?: boolean;
}
```

Implements `ControlValueAccessor` for form integration.
